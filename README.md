### 一、简介
本文档描述了支付平台（以下简称平台）与合作伙伴平台（以下简称客户端）之间的接口交互规范，这些接口用于网银支付、支付宝等充值请求以及订单状态查询、订单状态同步等规范。
### 二、接口说明
#### 2.1 传输协议
* 平台与客户端的通讯交互，所有协议使用HTTP传输。
* HTTP响应头中的Content-Type和Body内容格式参考接口说明里的接口规范
